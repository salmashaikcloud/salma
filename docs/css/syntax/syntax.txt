How to write a CSS code?
    A CSS consists of a selector and a declaration block.
    Where selector points to the HTML elements you want to style.
    Declaration block contains one or more-declarations,
    seperated by a semicolon ;
    Each decleration includes a CSS property-name and property-value,
    seperated by a colon[:].
    Declaration block is surrounded by curly braces.
--
Syntax:
selector {
    property-name1 : property-value1;
    property-name2 : property-value2;
    property-name3 : property-value3;
}
--
Example:
h3 {
    color: red;
    background-color: green;
    border: 1px solid red;
    }
--
