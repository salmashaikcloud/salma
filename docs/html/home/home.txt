What is Language?
	A language is a structured system of communication used by humans.
	Languages can be based on speech, gesture, sign, or writing.
	The structure of language is its grammar and the free components are its 
	vocabulary.
--
What is Markup Language?
	Marking up the text within the tags is called markup language.
Example:
	<imp>Marking up the text within the tags is called markup language.</imp>
Example of markup languages
	HTML, XML, XSLT,PHP and etc....
--
What is Hyper Text?
	Hypertext is a text with hyperlinks, when you click on hyperlinks, it will 
	navigates to other page.
	Hypertext documents are interconnected by hyperlinks, which are typically 
	activated by a mouse click, keypress, or screen touch.
--
What is HTML?
	HTML stand's for Hyper Text Markup Language.
	It is a tag based programming language.
	It is the only language to create a webpage.
	It describes the structure of a Web page.
	It consist of many tags/elements (approx 100).
	Elements tell the browser how to display the content.
