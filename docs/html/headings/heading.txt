HTML headings
    Headings are title or subtitles that you want to display on a web page.
    Headings are defined with the <h1> to <h6> tags.
    <h1> tag defines the most important heading and <h6> tag defines the least 
	important tag.
Example:
    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
    <h4>Heading 4</h4>
    <h5>Heading 5</h5>
    <h6>Heading 6</h6>
--
Bigger Headings
    Each HTML Heading has a default size. However You can specify the size of
    an Heading by using style attribute, using the CSS font-size property:
Example:
    <h1 style="font-size:80px">This is a heading</h1>
    You can use style attribute to change the font size of an heading.
