What are the difference between input type="submit", "button" and "reset"?
    submit:
        submits the form to middleware.
    button:
        it calls JavaScript.
    reset:
        it resets to its default value.

What is the difference between input type="radio" and "checkbox"?
    radio:
        we can select only one of many.
    checkbox:
        we can select zero to many.
