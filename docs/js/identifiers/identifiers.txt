What are Identifiers?
    All JavaScript variable-names, method-names, class-names must be identified 
	with unique-names.
    These unique-names are called identifiers.
    Identifiers can be short names (like x and y) or more descriptive names 
	(age, sum, totalVolume).
--
What are the rules for declaring Identifiers?
    Contain letters (a-z,A-Z), digits (0-9), underscores(_) and dollar($) signs.
    Can't start with digit.
	Can start with special characters _, $.
	Case sensitive.
	No length limit.
	reserved words are not allowed.
--
Note:
    JavaScript identifiers are case-sensitive.
